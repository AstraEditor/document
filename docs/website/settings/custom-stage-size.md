# 自定义舞台大小

将 Scratch 的舞台尺寸从 480x360（4:3 比例）调整为其他尺寸。

!!! warning "注意啦！"
    大多数作品在使用自定义舞台大小后都无法正常工作。

640x360 将形成一个宽屏 16:9 的画面。当然，你也可以使用更高的分辨率，如 1280x720、1920x1080 或更大尺寸的分辨率，这在技术上是可行的，但我们建议使用相同的宽高比但较低的分辨率，并通过全屏模式将其放大至用户的显示器上（要知道，有些用户的显示器可能无法容纳如此大的画面！）

作品可以通过像 https://scratch.mit.edu/projects/588493413/ 的方法来检测舞台大小。
