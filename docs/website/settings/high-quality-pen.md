# 高清画笔

高清画笔使得画笔作品能够以更高的分辨率进行渲染，而不再局限于 480x360(舞台分辨率) 的分辨率，并且还消除了编辑器中的某些坐标取整功能。

!!! warning "注意啦！"
    高清画笔的确能够显著提升画面表现效果，尤其是在全屏模式下，因为此时需要绘制的像素数量要多得多。<br>可是并非所有作品在启用高质量绘图工具后都会看起来更好。
